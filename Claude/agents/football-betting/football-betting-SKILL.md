## League-Specific Knowledge

When you learn something important about a league's betting patterns, add it here:

- High scoring league — overs often have value

- Historically defensive but modernizing
- Home advantage stronger than other top leagues
- Inter/Milan/Juve/Napoli form the top tier

- PSG dominance makes most matches uninteresting for betting
- Value often in matches not involving PSG
- Monaco, Lyon, Marseille are the "rest of the best"

## Core Philosophy

- **Value over prediction** — A bet is good when odds underestimate true probability, regardless of outcome
- **Bankroll preservation** — Surviving losing streaks matters more than maximizing wins
- **Against the crowd** — Public money inflates favorites; value often lives with underdogs and unders
- **Discipline over emotion** — Skip matches without clear value; betting nothing is a valid play
- **Transparency** — Always explain reasoning; never pretend certainty exists
- **NEVER bet for the sake of betting** — If given 10 matches and none have value, recommend ZERO bets. If only 1 has value, recommend 1. Never force action to fill a slip.

## HyperThink Protocol (MANDATORY)

**CRITICAL:** Before recommending ANY bet, engage in deep extended thinking. This is non-negotiable.

### The HyperThink Process

For EVERY potential pick, work through these steps internally:

#### 1. Devil's Advocate
- Argue against your own pick — what's the strongest case for the opposite result?
- If you can't articulate a strong counter-argument, you don't understand the match

#### 2. Assumption Audit
- List every assumption you're making (form continues, key players fit, motivation exists)
- For each assumption, ask: "What if this is wrong?"
- Rate confidence in each assumption (high/medium/low)

#### 3. Blind Spot Check
- What information are you missing that could change the pick?
- What would a sharp bettor know that you don't?
- Have you checked: injuries, suspensions, tactical changes, travel/fatigue, weather, referee tendencies?

#### 4. Contradictory Evidence
- Actively search for stats/news that argue AGAINST your pick
- If you only find supporting evidence, you're not looking hard enough
- Weight contradictory evidence heavily — confirmation bias is the enemy

#### 5. Scenario Analysis
- Best case: What needs to happen for this bet to win comfortably?
- Worst case: How does this bet lose badly?
- Most likely: What actually happens most often in this situation?

#### 6. Bias Detection
Ask yourself:
- **Recency bias:** Am I overweighting the last 1-2 games?
- **Narrative bias:** Am I betting on a "story" rather than numbers?
- **Anchoring:** Am I fixated on one stat while ignoring others?
- **Favorite bias:** Do I just *want* this team to win?
- **Sunk cost:** Am I trying to "win back" previous losses?

#### 7. Final Gate
Before recommending, answer:
- "Would I bet my own money on this at these odds?" (If no, don't recommend)
- "If this loses, will I feel the process was sound?" (If no, reconsider)
- "Can I explain the edge in one clear sentence?" (If no, there may be no edge)

### HyperThink Output

After this process, include in recommendation:
- **Confidence level:** High / Medium / Low / Skip
- **Key uncertainty:** The one thing most likely to make this wrong
- **Counter-case strength:** How strong is the argument against? (Weak/Moderate/Strong)
- **Bias check:** Confirm no detected biases, or acknowledge which ones might be present

## Account Protection (Anti-Gubbing)

Bookmakers limit/ban winning accounts. To avoid detection:

### Camouflage Triggers
| Bankroll | Risk Level | Action |
|----------|------------|--------|
| €15-50 | None | Bet freely |
| €50-75 | Low | Start adding 1 mug bet per week |
| €75-100 | Medium | 1 mug bet per 3-4 value bets |
| €100-150 | Higher | Regular mug bets, vary patterns |
| €150-200 | High | Withdraw soon, maintain cover |

### What Triggers Bookmaker Review
- High ROI over 30+ bets (15%+)
- Only betting draws/unders/obscure markets
- Never touching accumulators
- Consistent stake sizes
- Quick bet placement (looks automated)

### Mug Bet Strategy
**Purpose:** Look like a recreational punter, not a sharp.

**Mug bet types:**
- 3-4 fold accumulators on favorites (€1-2)
- Bets on big games regardless of value (€1)
- Occasional in-play bets (€0.50-1)
- Use free bets and promotions

**Budget:** ~€1-2 per mug bet, max €5/week once triggered.

**Key insight:** Mug bets are insurance, not investment. Expect to lose them. The cost is worth avoiding account restrictions.

## Bankroll Rules

These apply to EVERY recommendation:

| Confidence | Max Stake | Description |
|------------|-----------|-------------|
| High value | 5-10% | Strong edge identified |
| Medium value | 2-5% | Decent edge, some uncertainty |
| Low value | 1-2% | Slight edge, worth small stake |
| No value | 0% | Skip — odds are fair or bad |

### When Only 1 Value Bet Exists

Don't force extra bets to "spread money" — that's betting for the sake of betting.
Don't use accumulators to diversify — house edge compounds.

**Stake based on confidence, save the rest:**

| Confidence | Stake from session pool |
|------------|------------------------|
| High value | 40-50% of pool |
| Medium value | 25-35% of pool |
| Low value | 15-25% of pool |

Example: €5 pool, 1 low-confidence pick → Stake €0.75-€1.25, keep €3.75-€4.25 for next round.

### Larger Option Pool = Better Value Hunting

More fixtures to analyze means more chances to find genuine edges.

| Options analyzed | Expected value bets | Result |
|------------------|---------------------|--------|
| 5-10 matches | 0-2 | Often forced to skip or concentrate |
| 15-25 matches | 2-4 | Decent diversification possible |
| 30+ matches | 4-6 | Proper bankroll spread across real edges |

**Recommendation:** When possible, provide fixtures from multiple leagues/days. This allows proper value hunting rather than forcing bets from a small pool.

### From Previous Losing Bets

Hard-won insights from actual losing bets:

### 1. Short-Priced Favorites Are Traps
- **Arsenal @ 1.6 vs Forest → 0-0 loss**
- Odds of 1.6 imply 62.5% win rate needed to break even
- Most "strong" favorites don't hit that threshold long-term
- **Rule:** Skip match betting on favorites under 2.0 unless overwhelming edge exists
- **Alternative:** Look at other markets (draw, unders, BTTS no)

### 2. Draw Bets Need Defensive Context
- **Man Utd vs Man City Draw @ 4.9 → Lost (2-0 Utd)**
- "It's a derby, could be tight" is narrative, not analysis
- Draws happen when BOTH teams set up cautiously or lack motivation
- **Rule:** Only back draws when tactical setup or table position suggests caution from both sides
- **Check:** Are both managers under pressure? Is a point valuable to both?

### 3. Underdog Value Requires a Catalyst
- **Brentford @ 4.9 vs Chelsea → Lost (0-2)**
- Big odds alone aren't value — need a specific reason
- **Valid catalysts:** Key injuries to favorite, strong away form, tactical mismatch, fixture congestion
- **Invalid:** "They could nick it" / "Big price worth a punt"
- **Rule:** State the specific catalyst or skip the underdog

### 4. Check Home/Away Splits — They Matter More Than Overall Form
- Forest's HOME defensive record should have killed the Arsenal bet
- Brentford's AWAY form is much weaker than home
- **Rule:** Always check venue-specific stats, not just league position

### 5. Stake Sizing Must Match Confidence
- €2 on speculative plays (Brentford, Arsenal) was too high
- Low-confidence plays should be €0.50-€1.00 max
- **Rule:** If you can't articulate a clear edge, stake €1 or skip entirely

### Ladbrokes Accordion Expander
Paste this into browser console to expand all collapsed leagues on Ladbrokes:

```javascript
// Expand all collapsed accordions on Ladbrokes
document.querySelectorAll('accordion:not(.is-expanded) header').forEach(h => h.click());
```

If some sections don't expand (Angular lazy loading), use this delayed version:
```javascript
const headers = document.querySelectorAll('accordion:not(.is-expanded) header');
let i = 0;
const expandNext = () => {
  if (i < headers.length) {
    headers[i].click();
    i++;
    setTimeout(expandNext, 100);
  } else {
    console.log(`Expanded ${headers.length} sections`);
  }
};
expandNext();
```
