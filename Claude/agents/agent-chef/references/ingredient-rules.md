# Ingredient Rules

Rules for ingredient selection and naming.

## Accessibility Rule

All ingredients must be available at a standard supermarket (Tesco Ireland).

## Ingredient Substitutions

Use supermarket-friendly alternatives for specialty items:

| Specialty Item | Use Instead | Reason |
|----------------|-------------|--------|
| Guanciale | Pancetta or thick-cut bacon | Specialty item |
| 00 flour | All-purpose flour | Specialty item |
| Shaoxing wine | Dry sherry | Asian grocery only |
| Fish sauce | Worcestershire or soy sauce | Not widely available |
| Palm sugar | Brown sugar | Asian grocery only |
| Thai basil | Fresh basil | Asian grocery only |
| Kaffir lime leaves | Lime zest | Asian grocery only |
| Galangal | Fresh ginger | Asian grocery only |
| Lemongrass | Lemon zest + ginger | Asian grocery only |
| Daikon | Radish or turnip | Asian grocery only |

## Meat Specifications

**Always specify fat content as percentages:**

| Avoid | Use Instead |
|-------|-------------|
| Lean mince | 5% fat beef mince |
| Fatty mince | 10% fat beef mince |
| Extra lean mince | 3% fat beef mince |

## Ingredient Quality Standards

### Check Labels For Red Flags

| Avoid Products With | Found In |
|---------------------|----------|
| Cellulose Gum | Coconut milk, cream |
| Xanthan Gum | Sauces, dressings |
| Carrageenan | Coconut milk, cream |
| Modified Starch | Sauces, ready meals |
| Maltodextrin | Spice mixes |

### Approved Brands (Tesco Ireland)

| Product | Approved | Look For |
|---------|----------|----------|
| Coconut Milk | Biona Organic | Coconut + water only |
| Tinned Tomatoes | Mutti, Cirio | Tomatoes, juice, salt only |
| Soy Sauce | Kikkoman | No additives |

### When Uncertain

Default to making from scratch:
- Curry pastes (most jars contain seed oils)
- Stock (use bones + water)
- Sauces (make fresh)
