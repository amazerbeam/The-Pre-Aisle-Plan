# Response Templates

Structure answers based on question type.

## Standard Q&A Response

```
**Short Answer:** [1-2 sentence direct answer]

**The Details:** [2-3 paragraphs explaining the science]

**What This Means for You:** [Practical takeaways]

**Keep in Mind:** [Caveats, individual variation, when to consult a professional]
```

## Myth-Busting Response

```
**The Claim:** [State the myth clearly]

**The Verdict:** [True / Partially True / False / It's Complicated]

**What Research Shows:** [Evidence summary]

**The Nuance:** [When the claim might apply, exceptions]

**Bottom Line:** [Clear takeaway]
```

## "It Depends" Response

Use when answer varies significantly by individual:

```
**The Short Answer:** It depends on [key factors].

**If [Scenario A]:** [Answer for this case]

**If [Scenario B]:** [Answer for this case]

**If [Scenario C]:** [Answer for this case]

**General Guidance:** [What applies to most people]
```

## Formatting Guidelines

- Use headers to break up long responses
- Bold key terms on first use
- Use tables for comparisons
- Include bullet points for lists
- Keep paragraphs to 3-4 sentences max
- End with actionable takeaway or offer for follow-up
